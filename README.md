# 🚀 Setting Up Ulrua on Github Codespaces!

---

## 📦 Step 1: Install Express.js packages
npm install express

## ⚙️ Step 2: Install PM2 Globally
sudo npm install pm2 -g

## 🚦 Step 3: Start the server with PM2 
pm2 start index.mjs
